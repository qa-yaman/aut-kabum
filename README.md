# aut-kabum
Projeto de Automacao Java - Ecommerce Kabum

## Estrutura do Projeto



```
src
└── main
└── test
    ├── helpers
    ├── pages
    ├── runners
    ├── steps
    └── resources
        └── features
```

## Como Executar

```
mvn clean test
```

Para visualizar relatórios do Allure:

```
allure serve target/allure-results
```
